# Генератор случайных паролей

**Автор:** Зянкин Иван

## Короткое описание программы

Программа "Random Password Generator" представляет собой графическое приложение для генерации надежных случайных паролей с возможностью настройки параметров. Приложение позволяет пользователю выбирать длину пароля и наборы символов (цифры, буквы, спецсимволы), автоматически генерировать пароли, копировать их в буфер обмена и сохранять историю всех сгенерированных паролей в JSON-файл.

### Основные возможности:
- Настройка длины пароля от 4 до 64 символов с помощью ползунка
- Выбор типов символов через чекбоксы (цифры, буквы, спецсимволы)
- Генерация пароля с гарантированным включением выбранных типов символов
- Копирование пароля в буфер обмена
- История всех сгенерированных паролей с датой и временем
- Сохранение и загрузка истории в JSON-файл
- Удаление отдельных записей или всей истории
- Валидация ввода с проверкой минимальной длины

## Инструкции по запуску

### Требования к системе:
- Python 3.6 или выше
- Стандартные библиотеки Python (установлены по умолчанию):
  - `tkinter` (для графического интерфейса)
  - `random` (для генерации паролей)
  - `string` (для работы со строками)
  - `json` (для сохранения истории)
  - `os` (для работы с файлами)
  - `datetime` (для отметок времени)

### Установка зависимостей:

Дополнительных зависимостей не требуется, так как используются только стандартные библиотеки Python.

**Для Linux (Ubuntu/Debian):** может потребоваться установка tkinter:
```bash
sudo apt-get install python3-tk
```

**Для Windows и macOS:** tkinter обычно уже установлен вместе с Python.

### Запуск программы:

1. Сохраните код программы в файл `password_generator.py`

2. Откройте терминал (командную строку)

3. Запустите программу командой:
```bash
python password_generator.py
```

или если используется Python 3:
```bash
python3 password_generator.py
```

### Альтернативный способ запуска:
Некоторые системы требуют указания полного пути:
```bash
C:\Python39\python.exe password_generator.py
```

## Примеры работы и использования программы

### Пример 1: Генерация простого пароля

**Действия пользователя:**
1. Установить длину пароля: 8
2. Выбрать: только "Буквы"
3. Нажать "Сгенерировать пароль"

**Результат:** `aBcDeFgH` (пример)

### Пример 2: Генерация сложного пароля

**Действия пользователя:**
1. Установить длину пароля: 16
2. Выбрать: "Цифры", "Буквы", "Спецсимволы"
3. Нажать "Сгенерировать пароль"

**Результат:** `K#9mP$2vL@5qR!8n` (пример)

### Пример 3: Копирование пароля и работа с историей

**Последовательность действий:**
1. Сгенерировать пароль
2. Нажать кнопку "📋 Копировать"
3. Пароль скопирован в буфер обмена (появляется уведомление)
4. Сгенерировать еще несколько паролей (все сохраняются в истории)
5. Дважды кликнуть по любой записи в истории для копирования старого пароля
6. Нажать "❌ Удалить выбранное" для удаления ненужных записей

### Пример 4: Обработка ошибок

**Ситуация:** Пользователь пытается сгенерировать пароль без выбора символов

**Результат:** Появляется окно с ошибкой: "Не выбран ни один тип символов! Выберите хотя бы один вариант"

**Ситуация:** Длина пароля меньше количества выбранных наборов символов

**Результат:** Появляется окно с ошибкой: "Недостаточная длина пароля! Для выбранных наборов символов требуется длина не менее X символов"

### Пример сохранения данных

При закрытии программы появляется диалог: "Сохранить историю перед выходом?" При подтверждении все пароли сохраняются в файл `password_history.json` в следующем формате:

```json
[
  {
    "timestamp": "2026-05-05 14:30:15",
    "password": "K#9mP$2vL@5qR!8n",
    "length": 16
  },
  {
    "timestamp": "2026-05-05 14:28:42",
    "password": "aBcDeFgH",
    "length": 8
  }
]
```

При следующем запуске программа автоматически загрузит сохраненную историю.

### Скриншот интерфейса (описание)

Интерфейс программы включает:
- **Верхняя панель:** Ползунок длины пароля с отображением текущего значения
- **Средняя панель:** Три чекбокса для выбора типов символов
- **Панель генерации:** Поле для отображения пароля и кнопки "Сгенерировать" и "Копировать"
- **Нижняя панель:** Таблица истории с колонками "Дата и время", "Пароль", "Длина" и кнопками управления
- **Статусная строка:** Подсказка о двойном клике для копирования из истории

## Примечания

- Программа автоматически корректирует минимальную длину пароля в зависимости от выбранных наборов символов
- Все пароли генерируются с использованием библиотеки `random` для обеспечения случайности
- Максимальная длина пароля ограничена 64 символами для обеспечения удобства использования
- История сохраняется автоматически после каждой генерации и при закрытии программы
